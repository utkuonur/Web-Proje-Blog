# Full-Stack Blog Uygulaması

Bu proje, NestJS (Backend) ve React (Frontend) kullanılarak geliştirilmiş kapsamlı bir blog uygulamasıdır.

### Özellikler
- **Kullanıcı Sistemi:** Kayıt, Giriş ve Rol yönetimi (Admin/User).
- **Yazı Yönetimi:** CRUD işlemleri (Ekleme, Silme, Güncelleme, Listeleme).
- **Kategori Sistemi:** Many-to-Many ilişkisi ile yazıların kategorize edilmesi.
- **Beğeni Sistemi:** Yazıları beğenme/beğeniyi geri çekme özelliği.
- **Veritabanı:** SQLite & TypeORM ile ilişkisel veri modeli.

### Kurulum ve Çalıştırma
1. Repo'yu klonlayın: `git clone ...`
2. Backend: `cd backend && npm install && npm run start:dev`
3. Frontend: `cd frontend && npm install && npm run dev`
